# Fashion Store

Прототип веб-витрины магазина одежды: категории, карточки товаров, модальное окно с выбором цвета и размера, просмотр видео по товару, корзина, отправка текста заказа в Telegram и копирование для MAX.

## Структура проекта

```
├── index.html           # точка входа
├── css/
│   └── styles.css       # стили
├── js/
│   ├── app.js           # логика интерфейса
│   ├── products.js      # данные каталога
│   └── config.js        # Telegram и прочие настройки
├── assets/
│   ├── images/          # фото товаров
│   └── videos/          # ролики для превью (опционально)
├── package.json
└── README.md
```

## Запуск локально

ES-модули не открываются с диска через `file://` — нужен локальный сервер:

```bash
npm install
npm start
```

Откройте в браузере указанный адрес (обычно `http://localhost:3000`).

Без установки зависимостей:

```bash
npx serve . -l 3000
```

## Настройка

1. **Telegram:** в файле `js/config.js` замените `TELEGRAM_USERNAME` на ваш логин без символа `@`.
2. **Каталог:** правьте массив `products` и при необходимости `categories` в `js/products.js`.
3. **Медиа:** положите файлы в `assets/images/` и `assets/videos/` или укажите внешние URL в `products.js`.
